제품 요구사항 문서 (PRD)

제품명: SHE 시나리오 기반 디지털 훈련 플랫폼

버전: v1.0 (MVP 기준)

작성일: 2025.09.17

작성자: [작성자 이름]

1. 개요

목적: 실제 지역 기반 사고 시나리오를 바탕으로 각 개인의 역할(R&R)에 따른 조치 수행 능력을 점검, 평가, 리포팅할 수 있는 디지털 훈련 플랫폼 구축.

핵심 기능: 시나리오 기반 훈련, 자동 R&R 할당 및 대체자 매칭, 상황 전개 시뮬레이션, 행동 평가 시스템, 반복 훈련 및 리마인더 기능 포함

적용 대상: 기업의 안전관리팀, 지역관리팀, 홍보실 등 SHE 관련 실무자 전반

2. 주요 기능 요약
기능명	설명
시나리오 기반 훈련 콘텐츠	실제 사례 기반의 상황 구성, 변수 포함 시나리오 제공
R&R 매칭 인터페이스	조직도 기반 자동 역할 할당 및 결원 시 대체 인력 지정
시뮬레이션 엔진	시간 흐름에 따른 상황 전개, 사용자 선택 및 협업 기능 제공
평가 및 리포팅 시스템	행동 적절성/속도/정확도 등 정량 평가, 개인/부서별 리포트 생성
정기 훈련 기능	자동 푸시 알림, 모바일/PC 대응, 과제 기반 훈련 포함
3. 사용자 시나리오 예시

사건: OO동 OOO아파트 인근 도시가스 중압배관 파손 (09:35, 평일)

경보 수신 → 관제센터 알림

현장 점검 여부 결정 (R&R: 관제운영팀)

주민 대피 방송 판단 (R&R: 지역관리/안전관리팀)

언론 대응 결정 (R&R: 홍보실/경영진)

인원 부재 → 대체 인력 자동 지정

종료 후 리포트 자동 생성

4. 상세 기능 명세
4.1. 상황 시나리오 콘텐츠

지역/사건 기반 템플릿 구성 (PDF or DB 형식)

시나리오 변수: 사고 시각, 날씨, 통보 방식, 외부기관 협조 등

사용자 입력: 행동 선택, 문서 작성, 의사결정 기록

4.2. R&R 매칭 인터페이스

조직도 연동 자동 역할 매칭

훈련 중 인원 부재 → 대체자 자동 지정 + 푸시 알림

각 역할별 책임 명시 및 판단 포인트 사전 설정

4.3. 훈련 상황 시뮬레이션 엔진

시간 흐름 따라 상황 자동 전개 (타임라인 기반)

시점별 사용자 조치 입력 필요

협업 기능: 실시간 보고, 결재, 협조 요청 등

4.4. 평가 및 리포팅

평가 요소: 반응 속도, 선택 적절성, 커뮤니케이션 정확성 등

점수화 기준: 리더십/협업/정확도 기반 스코어링

결과 리포트: 개인별, 부서별, 전체 리포트 자동 생성

4.5. 반복 훈련 및 리마인드

정기 훈련 푸시 알림 (월간/분기별)

PC/모바일 모두 대응

특정 상황에 대한 체크리스트 기반 과제 포함

5. 기술 스택 및 플랫폼 고려사항 (MVP 기준)

프론트엔드: React 또는 Vue 기반 SPA

백엔드: Node.js + Express 또는 Firebase 기반 서버리스

DB: Firebase Firestore / PostgreSQL

배포: 사내 인트라넷 또는 AWS 기반 클라우드

MVP 프로토타입: Google Form + Excel 자동화 + PDF 시나리오 활용

6. 확장 계획

1단계: 5개 시나리오 템플릿 제작 (지역 기반)

2단계: 역할별 조치 매뉴얼 설계 및 매핑

3단계: Excel 기반 자동 평가표 개발

4단계: 웹 프로토타입 외주 개발 요청

5단계: 부서 내 테스트 후 전사 확대 적용

7. 기대 효과
항목	기대 효과
반복 훈련	실제 사고 시나리오 기반으로 반복 학습 가능
역할 인지	개인 R&R + 타인 역할 흐름 파악 가능
협업 훈련	유관 부서간 절차적 협업 강화
대응력 향상	실제 사고 발생 시 신속 대응 가능
법적 대응	훈련 이력 자동 저장 → 법적 대응자료 활용 가능
8. 기타 고려 사항

UI/UX 직관성 확보: 직책/경력 무관하게 누구나 접근 가능해야 함

보안: 사내 보안 정책에 따라 로그인/접속 제한 기능 필요

확장성: 향후 다양한 유형의 시나리오 및 역할 추가 가능 구조 필요
