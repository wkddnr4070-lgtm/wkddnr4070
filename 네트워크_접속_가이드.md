## ⚠️ 주의사항

### 보안
- 현재는 개발용 설정으로 보안이 완전하지 않습니다
- 실제 운영 시에는 방화벽 설정이 필요합니다

### 네트워크 요구사항
- 모든 기기가 같은 네트워크에 있어야 합니다
- 방화벽이 포트 3000, 3001을 차단하지 않아야 합니다

### 성능
- 동시 접속자 수가 많으면 성능이 저하될 수 있습니다
- 권장 동시 접속자: 5-10명

## 🛠️ 문제 해결

### 접속이 안 될 때
1. **방화벽 확인**
   - Windows 방화벽에서 포트 3000, 3001 허용
   
2. **네트워크 확인**
   - 같은 Wi-Fi 네트워크인지 확인
   - IP 주소가 변경되지 않았는지 확인

3. **서버 상태 확인**
   - 서버가 실행 중인지 확인
   - 터미널에서 오류 메시지 확인

### IP 주소 변경 시
```bash
# 새로운 IP 주소 확인
ipconfig

# backend/src/config/index.js 파일에서 CORS 설정 업데이트
```
