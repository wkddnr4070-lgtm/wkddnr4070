# 🚀 초보자를 위한 ngrok 공용 접속 설정 (초간단 버전)

## 📋 준비물
- 컴퓨터 (Windows)
- 인터넷 연결
- 이메일 주소 (ngrok 계정용)

## 🎯 목표
- 다른 사람이 인터넷에서 우리 사이트에 접속할 수 있게 만들기

---

## 📝 1단계: ngrok 계정 만들기 (5분)

### 1-1. 웹사이트 접속
1. **웹브라우저 열기** (Chrome, Edge, Firefox 등)
2. **주소창에 입력**: `https://ngrok.com`
3. **Enter 키 누르기**

### 1-2. 계정 생성
1. **"Sign up" 버튼 클릭** (회원가입)
2. **이메일 입력**: 본인 이메일 주소
3. **비밀번호 입력**: 기억하기 쉬운 비밀번호
4. **"Sign up" 버튼 클릭**

### 1-3. 이메일 확인
1. **이메일함 확인**
2. **ngrok에서 온 이메일 찾기**
3. **"Verify Email" 클릭**

---

## 🔑 2단계: 인증 토큰 받기 (2분)

### 2-1. 로그인
1. **ngrok.com에서 "Log in" 클릭**
2. **이메일과 비밀번호 입력**
3. **"Log in" 클릭**

### 2-2. 토큰 복사
1. **대시보드 화면에서 "Your Authtoken" 찾기**
2. **긴 글자 복사** (예: `2abc123def456ghi789jkl012mno345pqr678stu901vwx234yz567890`)
3. **복사한 토큰 메모장에 저장** (나중에 필요)

---

## 🖥️ 3단계: 컴퓨터에서 설정하기 (3분)

### 3-1. 프로젝트 폴더 열기
1. **파일 탐색기 열기**
2. **"C:\Users\com\Desktop\SHE 디지털트윈" 폴더로 이동**
3. **폴더 안에 있는 파일들 확인**

### 3-2. 자동 설정 파일 실행
1. **"ngrok_문제해결_설정.bat" 파일 찾기**
2. **파일을 더블클릭** (두 번 클릭)
3. **검은 창이 열리면 기다리기**

### 3-3. 인증 토큰 입력
1. **"인증 토큰을 입력하세요:" 라는 글자 보기**
2. **복사한 토큰 붙여넣기** (Ctrl+V)
3. **Enter 키 누르기**

---

## 🌐 4단계: 공용 URL 받기 (2분)

### 4-1. ngrok 실행 확인
1. **새로운 검은 창이 열림**
2. **화면에 긴 글자들이 나타남**
3. **"Forwarding" 이라는 줄 찾기**

### 4-2. 공용 URL 복사
1. **"Forwarding" 줄에서 "https://" 로 시작하는 주소 찾기**
2. **예시**: `https://abc123.ngrok.io`
3. **이 주소를 복사** (Ctrl+C)

---

## 📱 5단계: 다른 사람에게 공유하기 (1분)

### 5-1. 공유할 정보 준비
```
접속 주소: https://abc123.ngrok.io
로그인 정보:
- 사용자명: dnrdl4070
- 비밀번호: @wlsghks12
```

### 5-2. 공유 방법
- **카카오톡**: 메시지로 전송
- **이메일**: 이메일로 전송
- **문자**: 문자로 전송
- **전화**: 전화로 알려주기

## ✅ 완료! 이제 다른 사람이 접속할 수 있습니다

### 🎉 성공 확인
1. **다른 사람이 주소로 접속**
2. **로그인 정보 입력**
3. **사이트 정상 작동 확인**

---

## 🚨 문제가 생겼을 때

### 서버가 안 켜질 때
1. **"서버_시작.bat" 파일 더블클릭**
2. **검은 창이 열리면 기다리기**
3. **"서버 시작 완료!" 메시지 확인**

### ngrok이 안 될 때
1. **"ngrok_8012_해결.bat" 파일 더블클릭**
2. **안내에 따라 진행**
3. **문제 해결 확인**

### 접속이 안 될 때
1. **서버가 켜져 있는지 확인**
2. **ngrok이 실행 중인지 확인**
3. **인터넷 연결 확인**
