# SHE 디지털 훈련 플랫폼 프로토타입

실제 지역 기반 사고 시나리오를 바탕으로 각 개인의 역할(R&R)에 따른 조치 수행 능력을 점검, 평가, 리포팅할 수 있는 디지털 훈련 플랫폼입니다.

## 🚀 주요 기능

### 1. 시나리오 기반 훈련
- 실제 사례 기반의 상황 구성
- 시간 흐름에 따른 상황 전개
- 대화형 의사결정 시스템
- 실시간 타임라인 추적

### 2. R&R 매칭 시스템
- 조직도 기반 자동 역할 할당
- 결원 시 대체 인력 자동 지정
- 역할별 책임 명시 및 관리
- 실시간 참여자 상태 추적

### 3. 평가 및 리포팅
- 반응 속도, 선택 적절성, 커뮤니케이션 정확성 평가
- 개인별, 부서별, 전체 리포트 자동 생성
- 시나리오별 성과 분석
- 개선 권장사항 제공

### 4. 관리 기능
- 직관적인 대시보드
- 훈련 진행률 실시간 모니터링
- 모바일 반응형 UI
- 다양한 리포트 내보내기

## 📋 시나리오 예시

**도시가스 중압배관 파손 사고**
- 발생 시간: 평일 09:35
- 위치: OO동 OOO아파트 인근
- 대응 조직: 관제운영팀, 안전관리팀, 지역관리팀, 홍보실
- 주요 절차: 현장 점검 → 주민 대피 방송 → 언론 대응

## 🛠️ 기술 스택

- **Frontend**: React 18, Tailwind CSS
- **Routing**: React Router v6
- **Icons**: Lucide React
- **Build Tool**: Vite
- **Styling**: Tailwind CSS + PostCSS

## 📦 설치 및 실행

### 1. 의존성 설치
```bash
npm install
```

### 2. 개발 서버 실행
```bash
npm run dev
```

### 3. 프로덕션 빌드
```bash
npm run build
```

### 4. 프로덕션 미리보기
```bash
npm run preview
```

### EvaluationReport.jsx
- 훈련 성과 평가 및 분석
- 부서별/개인별 리포트
- 개선 권장사항 제공
