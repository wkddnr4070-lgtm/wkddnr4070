# 🌍 SHE 디지털트윈 공용 접속 설정 가이드

## 🚀 방법 1: ngrok 사용 (추천)

### ngrok 설치 및 설정
1. **ngrok 다운로드**: https://ngrok.com/download
2. **회원가입**: 무료 계정 생성
3. **인증 토큰 설정**:
   ```cmd
   ngrok config add-authtoken YOUR_AUTH_TOKEN
   ```

### 사용 방법
1. **서버 시작**:
   ```cmd
   # 백엔드 서버 시작
   cd backend
   npm start
   
   # 새 터미널에서 프론트엔드 시작
   npm start
   ```

2. **ngrok 터널 생성**:
   ```cmd
   # 프론트엔드용 터널 (포트 3001)
   ngrok http 3001
   
   # 백엔드용 터널 (포트 3001) - 새 터미널에서
   ngrok http 3001
   ```

3. **공용 URL 확인**:
   - ngrok이 제공하는 URL (예: `https://abc123.ngrok.io`)
   - 이 URL을 다른 사람에게 공유

### 장점
- ✅ 무료
- ✅ HTTPS 지원
- ✅ 간단한 설정
- ✅ 실시간 로그 확인

### 단점
- ❌ 무료 버전은 세션당 8시간 제한
- ❌ URL이 매번 변경됨 (유료 버전에서 고정 가능)

## 🔧 방법 3: Cloudflare Tunnel

### 설정 방법
1. **Cloudflare 계정 생성**
2. **cloudflared 설치**:
   ```cmd
   winget install cloudflared
   ```

3. **터널 생성**:
   ```cmd
   cloudflared tunnel create she-training
   ```

4. **터널 실행**:
   ```cmd
   cloudflared tunnel --url http://localhost:3001 run she-training
   ```

### 장점
- ✅ 무료
- ✅ 고정 URL
- ✅ 빠른 속도
- ✅ 보안 강화
