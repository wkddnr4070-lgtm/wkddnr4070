# 로고 설치 안내

## 📋 로고 파일 설치 방법

### **1단계: 로고 이미지 파일 준비**
- SK 이노베이션 E&S 로고 이미지 파일을 준비하세요
- 권장 형식: PNG (투명 배경 권장)
- 권장 크기: 높이 48px 이상 (비율 유지)

### **2단계: 파일 위치**
로고 이미지 파일을 다음 위치에 저장하세요:

```
public/
└── sk-innovation-logo.png
```

### **3단계: 파일명 확인**
현재 코드에서 다음 파일명을 참조하고 있습니다:
- `/sk-innovation-logo.png`

### **4단계: 테스트**
1. 브라우저에서 로그인 페이지 접속
2. 좌측 상단에 로고가 표시되는지 확인
3. 로고가 표시되지 않으면 브라우저 콘솔(F12)에서 오류 확인

---

## 🔧 대체 파일명 사용 방법

다른 파일명을 사용하려면 다음 파일들을 수정하세요:

### **BackendLogin.jsx**
```jsx
<img 
  src="/your-logo-filename.png"  // 여기 수정
  alt="SK 이노베이션 E&S" 
  className="h-12 object-contain"
/>
```

### **UserProfile.jsx**
```jsx
<img 
  src="/your-logo-filename.png"  // 여기 수정
  alt="SK 이노베이션 E&S" 
  className="h-12 object-contain"
/>
```

## 📝 현재 구현 상태

### **적용된 페이지**
- ✅ `BackendLogin.jsx` - 백엔드 연결 로그인 페이지
- ✅ `UserProfile.jsx` - 프론트엔드 프로필 설정 페이지

### **로고 위치**
- 좌측 상단 (`absolute top-6 left-6`)
- 높이: 48px (`h-12`)
- 반응형: 모바일에서도 적절히 표시

### **Fallback 처리**
- 로고 파일이 없을 경우 기본 아이콘과 텍스트로 대체 표시
- 오류 없이 안전하게 동작

## 🎨 로고 스타일 커스터마이징

로고 크기나 위치를 변경하려면 다음 클래스를 수정하세요:

```jsx
// 현재 설정
<div className="absolute top-6 left-6">
  <img 
    className="h-12 object-contain"
  />
</div>

// 크기 변경 예시
<div className="absolute top-4 left-4">
  <img 
    className="h-16 object-contain"  // 더 크게
  />
</div>

// 위치 변경 예시
<div className="absolute top-8 left-8">
  <img 
    className="h-10 object-contain"  // 더 작게
  />
</div>
```
