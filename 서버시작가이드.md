# 🚀 SHE 디지털트윈 플랫폼 서버 시작 가이드

## 📋 방법 1: 배치 파일 사용 (추천)

1. **ultimate-server-start.bat** 파일을 더블클릭하세요
2. 자동으로 모든 검사와 서버 시작이 진행됩니다

## 📋 방법 2: 수동 실행

### 1단계: 명령 프롬프트 열기
- **Windows 키 + R** 누르기
- `cmd` 입력하고 엔터

### 2단계: 프로젝트 폴더로 이동
```cmd
cd /d "C:\Users\com\Desktop\SHE 디지털트윈"
```

### 3단계: 서버 시작
```cmd
npm run dev
```

## 📋 방법 3: VS Code 터미널 사용

1. VS Code에서 프로젝트 폴더 열기
2. **터미널 > 새 터미널** (Ctrl + `)
3. `npm run dev` 입력

## 🌐 접속 주소

서버가 시작되면 브라우저에서 접속:
- **http://localhost:3000**
- **http://127.0.0.1:3000**

## 🚨 문제 해결

### Node.js가 설치되지 않은 경우
- https://nodejs.org 에서 다운로드하여 설치

### 포트 3000이 사용 중인 경우
- 작업 관리자에서 다른 프로세스 종료
- 또는 다른 포트 사용: `npm run dev -- --port 3001`

### 권한 문제가 있는 경우
- 명령 프롬프트를 "관리자 권한으로 실행"

## ✅ 성공 확인

서버가 성공적으로 시작되면 다음과 같은 메시지가 표시됩니다:

```
VITE v4.x.x  ready in xxx ms

➜  Local:   http://localhost:3000/
➜  Network: http://xxx.xxx.xxx.xxx:3000/
```
