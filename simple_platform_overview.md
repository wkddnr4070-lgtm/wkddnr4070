# 도시가스 비상대응 모의훈련 플랫폼 - 간단 소개

## 🏢 플랫폼 개요

```mermaid
graph TD
    A[👤 직원] --> B[🏢 회사 선택]
    B --> C[📋 역할 배정]
    C --> D[🎯 훈련 시작]
    D --> E[📊 결과 분석]
    E --> F[📈 성과 향상]
    
    style A fill:#e1f5fe
    style B fill:#f3e5f5
    style C fill:#e8f5e8
    style D fill:#fff3e0
    style E fill:#fce4ec
    style F fill:#e0f2f1
```

## 🎯 핵심 기능

```mermaid
graph LR
    subgraph "📱 사용자 관리"
        A1[회사 조직도<br/>9개 도시가스 회사]
        A2[직원 프로필<br/>실명/직책 관리]
        A3[역할 배정<br/>11개 전문 반]
    end
    
    subgraph "🎮 훈련 시스템"
        B1[실시간 시뮬레이션<br/>타이머 기반]
        B2[맞춤형 시나리오<br/>역할별 가이드]
        B3[즉시 피드백<br/>점수 및 개선점]
    end
    
    subgraph "📊 데이터 관리"
        C1[훈련 이력<br/>개인별 기록]
        C2[성과 분석<br/>통계 및 리포트]
        C3[보고서 생성<br/>상세 평가서]
    end
    
    A1 --> B1
    A2 --> B2
    A3 --> B3
    B1 --> C1
    B2 --> C2
    B3 --> C3
    
    style A1 fill:#e3f2fd
    style A2 fill:#e3f2fd
    style A3 fill:#e3f2fd
    style B1 fill:#f1f8e9
    style B2 fill:#f1f8e9

    style C2 fill:#fce4ec
```

## 🚀 훈련 프로세스

```mermaid
flowchart TD
    Start([🚀 시작]) --> Login[👤 로그인<br/>회사/부서/직원 선택]
    Login --> Role[🎭 역할 배정<br/>11개 전문 반 중 배정]
    Role --> Brief[📖 브리핑<br/>상황 설명 및 역할 안내]
    Brief --> Train[🎯 훈련 실행<br/>실시간 시나리오 진행]
    Train --> Action[⚡ 행동 선택<br/>1개 선택 제한]
    Action --> Score[📊 즉시 평가<br/>점수 및 피드백]
    Score --> Next{다음 단계?}
    Next -->|예| Train
    Next -->|아니오| Result[🏆 최종 결과<br/>종합 점수 및 리포트]
    Result --> End([✅ 완료])
    
    style Start fill:#4caf50,color:#fff
    style Login fill:#2196f3,color:#fff
    style Role fill:#9c27b0,color:#fff
    style Brief fill:#ff9800,color:#fff
    style Train fill:#f44336,color:#fff
    style Action fill:#795548,color:#fff
    style Score fill:#607d8b,color:#fff
    style Result fill:#e91e63,color:#fff
    style End fill:#4caf50,color:#fff
```

## 📈 기대 효과

```mermaid
graph LR
    Before[❌ 기존 방식<br/>• 물리적 훈련<br/>• 시간/장소 제약<br/>• 높은 비용<br/>• 제한적 참여] --> After[✅ 디지털 플랫폼<br/>• 언제든 접근<br/>• 비용 절감<br/>• 대규모 참여<br/>• 데이터 기반 개선]
    
    After --> Benefit1[📈 훈련 효과 향상<br/>실시간 피드백으로<br/>즉시 개선점 파악]
    After --> Benefit2[💰 비용 절감<br/>물리적 훈련 비용<br/>대폭 절약]
    After --> Benefit3[📊 데이터 축적<br/>훈련 이력 분석으로<br/>체계적 관리]
    
    style Before fill:#ffebee
    style After fill:#e8f5e8
    style Benefit1 fill:#e3f2fd
    style Benefit2 fill:#fff3e0
    style Benefit3 fill:#f3e5f5
```
