# 🚨 파일 이름 오류 완전 해결 가이드

## 🔍 문제 원인
- **오류 메시지**: "파일 이름, 디렉터리 이름 또는 볼륨 레이블 구문이 잘못되었습니다."
- **원인**: 한글 경로 `C:\Users\com\Desktop\SHE 디지털트윈` 사용
- **문제**: Windows 배치 파일에서 한글 경로 처리 오류

## 🛠️ 해결 방법

### 방법 1: 자동 해결 (추천)
1. **`파일이름_오류_완전해결.bat` 더블클릭**
2. **자동으로 영문 경로로 프로젝트 복사**
3. **새 위치에서 서버 시작**
4. **완료!**

### 방법 2: 수동 해결
1. **프로젝트 폴더 복사**:
   - 원본: `C:\Users\com\Desktop\SHE 디지털트윈`
   - 복사: `C:\SHE`

2. **새 위치에서 서버 시작**:
   ```cmd
   cd /d "C:\SHE"
   npm run dev
   ```

## 🎯 단계별 해결 과정

### 1단계: 프로젝트 복사
- 한글 경로에서 영문 경로로 모든 파일 복사
- `C:\SHE\` 폴더 생성 및 파일 이동

### 2단계: 서버 재시작
- 새 위치에서 서버 시작
- 정상 작동 확인

### 3단계: ngrok 재설정
- 새 위치에서 ngrok 시작
- 터널 재연결

## 📋 생성된 해결 도구

### 🔧 자동 해결 도구
- **`파일이름_오류_완전해결.bat`**: 완전 자동 해결
- **`영문경로_서버시작.bat`**: 영문 경로에서 서버 시작

### 📖 사용 방법
